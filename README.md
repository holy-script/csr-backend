# CSR Backend

Backend Developer Intern Task, based on the CSR Initiative

## Installation

To install the necessary dependencies, run the following command:
```bash
npm install
```


## Usage

To start the application, run:
```bash
npm run build
```
and then
```bash
npm start
```

To start the application in development mode, run:    
```bash
npm run dev
```
